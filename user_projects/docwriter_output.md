# Task Manager CLI

A simple command-line interface task manager written in Python.

## Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* View all tasks
* View completed tasks
* View pending tasks

## Installation

To install the task manager, run the following command:
```bash
pip install -r requirements.txt
```

## Usage

To run the task manager, execute the following command:
```bash
python task_manager.py
```

## Commands

* `add <task_name>`: Add a new task
* `delete <task_id>`: Delete a task
* `complete <task_id>`: Mark a task as completed
* `all`: View all tasks
* `completed`: View completed tasks
* `pending`: View pending tasks
